طراحی یک لندینگ پیج بسیار حرفه‌ای، جذاب و مدرن برای معرفی و فروش آنلاین بیمه خودرو (بیمه شخص ثالث و بیمه بدنه).

🛠️ **استک تکنولوژی:**
- **Astro** - Static Site Generator (بهینه‌سازی SEO و سرعت)
- **React 19** - کامپوننت‌های تعاملی
- **TypeScript** - Type Safety
- **Tailwind CSS** + **Design Tokens** - سیستم طراحی یکپارچه
- **Jotai** - State Management برای فرم‌ها و داده‌ها
- **Vazirmatn** - فونت فارسی

🎨 **ساختار کلی صفحه:**
- **Mobile-First Design** (80% کاربران موبایل)
- طراحی کاملاً واکنش‌گرا (Responsive) برای دسکتاپ، تبلت و موبایل
- استفاده از Design Tokens موجود (رنگ‌های orange، brown، gray)
- بهینه‌سازی برای تبلیغات پیامکی (سرعت لود بالا)
- از تصاویر بهینه‌شده WebP/AVIF برای خودروهای ایرانی استفاده شود

🎯 **State Management با Jotai:**
```typescript
// Insurance Calculator Atoms
export const carTypeAtom = atom<string>('')
export const carYearAtom = atom<number>(new Date().getFullYear())
export const discountPercentAtom = atom<number>(0)
export const plateNumberAtom = atom<string>('')
export const phoneNumberAtom = atom<string>('')

// UI State Atoms
export const activeTabAtom = atom<'third-party' | 'comprehensive'>('third-party')
export const isCalculatingAtom = atom<boolean>(false)

// Contact Form Atoms
export const contactFormAtom = atom({
  name: '',
  phone: '',
  message: ''
})

// Derived Atoms
export const calculationResultsAtom = atom(
  (get) => {
    const carType = get(carTypeAtom)
    const carYear = get(carYearAtom)
    const discount = get(discountPercentAtom)
    // محاسبه قیمت بر اساس داده‌ها
    return calculateInsurancePrice(carType, carYear, discount)
  }
)
```
